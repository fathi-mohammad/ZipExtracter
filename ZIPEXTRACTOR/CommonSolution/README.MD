This reposotory contain following .net  projects:
	1. zipprocessor.worker  -- main worker service application to handle zip extracted files.
	2.zipprocessor.Admin   -- Admin web portal to manage the worker service.
	3.zipprocessing.WebApi -- sample api to call during zip extracted file processing for worker service
	4.windowsformapp  -- sample window form application for zip extraction and log initial request details for testing purpose.
	5.workerservice1 -- another sample worker service application model , can be used for reference.

	Each project has its own README file for more details.
